# Start workspace governance

Run `k6s team start` with the user objective to begin governed workspace governance.

If the user provides an objective, use it directly:
`k6s team start "<objective>"`

If no objective is given, ask the user what they want to accomplish.

After starting, confirm the workspace governance ID and remind the user that all agent actions are audited.
